VSMC Single cell model

This model solves intracellular calcium dynamics in vascular smooth muscle cell (VSMC). 

How to Compile

make -f makefile

It will create run file and all the doxygen files. The doxygen files are stored in "doc" folder.

How to Run

./VSMCsinglecell
